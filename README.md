# MinimarketPN

Sito web vetrina per il Minimarket Piccolini.

## Funzionalità
- **Stato in tempo reale**: Mostra se il negozio è attualmente aperto o chiuso, calcolando dinamicamente il tempo rimanente alla prossima apertura o chiusura.
- **Informazioni sui Reparti**: Presenta i principali reparti del negozio (Prodotti Freschi, Dispensa, Bevande, Igiene e Casa).
- **Orari e Contatti**: Fornisce una tabella aggiornata degli orari di apertura settimanali e i recapiti del negozio.
- **Mappa Interattiva**: Integrazione con Google Maps per facilitare il raggiungimento della sede in Via G. Guibert 14, Caselle Torinese.

## Tecnologie utilizzate
- HTML5
- CSS3
- JavaScript (Vanilla JS)
